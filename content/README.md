# コンテンツ編集用CSV

アプリの初期データ(Can-Do項目・レベル目安・教材リスト)を、
Excel / Numbers / Googleスプレッドシートで編集するためのファイルです。

## 使い方(3ステップ)

1. `npm run content:export` — 最新のシードからCSVを書き出す(このフォルダに生成)
2. CSVを表計算ソフトで編集して**CSV形式のまま**上書き保存
3. `npm run content:import` — 検証つきで取り込み、アプリのシードに反映

取り込み後は `npm test` で確認し、アプリ側は「設定 → 初期テンプレートに戻す」
(または新しいブラウザ)で新データが見えます。編集済みCSVをそのまま
チャットでClaude に渡して反映を頼んでもOKです。

## ファイルと列

### cando.csv — Can-Do項目(1行=1項目)
| 列 | 内容 |
|---|---|
| 言語 | 日本語 / 英語 / 中国語 / 韓国語 / ポルトガル語 / スペイン語 |
| 段階 | 1〜6(S1入門〜S6上級) |
| 技能 | 聞く / 話す / 読む / 書く |
| 文言 | 観察できる行動で書く(例:「段階別リーダーを1冊読み切る」) |

- 行の追加・削除・文言修正は自由。同じ 言語×段階×技能 内の**行順=表示順**
- ⚠️ 既存項目の行順を入れ替えると内部IDが変わり、既存ユーザーのチェックとの対応がずれます。文言修正は行を動かさずその場で

### cells.csv — 段階ごとの目安(1行=言語×段階)
目安_外国語として(検定名)/ 目安_母語として(現地学年・基準)/
取り組みのヒント / 根拠メモ / 技能別の概要4列。36行(6言語×6段階)固定が前提です。

### materials.csv — 教材(1行=1教材)
| 列 | 内容 |
|---|---|
| 種類 | 本 / アプリ / オンライン / 動画 / ワーク |
| 言語・技能 | 複数は「;」区切り(例: `聞く;読む`) |
| 段階 | 「;」区切り(例: `2;3`)。範囲 `2-4` も可 |
| 対象者 | 子供 / 大人 / 共通 |
| 入手 | 日本 / 現地 |
| 内部ID | **編集しない**(既存ユーザーの教材ステータスが紐づく)。新規行は空欄でOK(自動採番) |

## 注意

- 文字コードはUTF-8(BOM付き)。Excelでそのまま開けます
- 値はラベル(日本語)でもコード(`ja` `listening` 等)でもOK
- 取り込み時に検証があり、エラーがあれば行番号つきで表示して中止します(既存データは壊れません)
