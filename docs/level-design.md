## 3. Can-Do項目の設計方針+サンプル

**方針:**
1. **年齢非依存の行動表現**で書く(「園での出来事を〜」ではなく「その日の出来事を〜」)。子供も大人も同じ項目で判定する
2. 項目は言語×段階×技能で共通。役割(母語/外国語)の違いは**ベンチマーク表示と教材推薦**にだけ反映する
3. 家庭で観察・自己確認できる具体的行動。1技能あたり2〜4項目
4. 検定合格も項目に含めてよいが、各技能に検定以外の項目を必ず併存させる

**サンプル(英語):**

- S3(初中級)読む:「段階別リーダー(ORT Stage 8〜9 / Lexile 400前後)を1冊読み切る」「短いメールや案内文の要点がわかる」「英検3級の長文問題で6割正解する」
- S3(初中級)話す:「身近な話題について理由を1つ添えて意見を言える」「日常の用事(注文・道案内)のやりとりができる」
- S5(中上級)聞く:「ニュースやドキュメンタリーの大意を字幕なしで掴める」「英検準1級リスニングで6割正解する」
- S5(中上級)書く:「120〜150語の構成の整った意見文を書ける」「仕事や学校の連絡メールを不自由なく書ける」

**サンプル(日本語・母語トラックでも外国語トラックでも同じ項目):**

- S2(初級)読む:「ひらがな・カタカナの短い本を1冊一人で読み切る」「基本の漢字(100字程度)を読める」
- S4(中級)書く:「600〜800字の意見文を構成立てて書ける」「引用と自分の考えを区別して書ける」
